## overview
### AI as a Communication Copilot
[download link]
![Placeholder](content/images/title.png)
ChatCopilot (CC) is an AI communication-assistant platform embedded in existing chat software. It doesn't chat for users, but helps them understand intentions, organize thoughts, and generate reply suggestions that match their own style.
![Placeholder](content/images/demo.gif)

## origin
### Origin: The Observation
![Placeholder](content/images/origin1.png)
The project originated from a three-year observation in a QQ drawing group, where many creators struggled to explain their work and get meaningful feedback.

## research
### PainPoint: Understanding & Expression
![Placeholder](content/images/research1.png)
User interviews revealed that the core difficulties in communication are not just about polishing words, but about understanding the other party's intent and properly expressing one's own ideas.
![Placeholder](content/images/research2.png)

## competition
### Why Existing Tools Aren't Enough
![Placeholder](content/images/competition1.png)
ChatBots, Writing Assistants, and Agents each have limitations. They either require users to leave the chat, only polish existing text, or lack the context for nuanced human communication.

## user_flow
### The User Flow
![Placeholder](content/images/userflow1.png)
From encountering a communication block, the user can perform a one-click analysis to understand the situation, review suggestions, and maintain full control over the final message.

## design
### Design Principle: AI as a tool
![Placeholder](content/images/design1.png)
The AI is responsible for analysis and suggestions, while the user is responsible for judging, modifying, and sending the final message.
![Placeholder](content/images/design2.png)

## feedback
### Iteration: Authentic Expression
![Placeholder](content/images/feedback1.png)
Initial user feedback was that AI suggestions were "helpful, but not me". To solve this, Language Style Memory was added to generate replies that better match the user's authentic voice.

## future

![Placeholder](content/images/future1.png)
